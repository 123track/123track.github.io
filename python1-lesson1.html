The Four Variables

In Python, there are four big variables. They are strings, integers, floats, and booleans.

If you have absolutely no idea about any of our vocabulary here, a variable is a value that well, varies. You can set a variable's data to just about anything that the type supports.

Integers might just be the most basic type to exist, with their only rule being that they are a whole number from negative infinity to positive infinity; they just cannot be a decimal.

Strings are essentially strings of text. Strings consist of a piece of text, surrounded by either single or double quotation marks. Now, I must assert my opinion here; double quotes are better. Some people prefer single quotes as they don't have to press shift while typing, but it really just makes a negligible difference.

Floats are integers, but with a massive upgrade. They are now able to accept decimal numbers.

Booleans are very simple, having only two possible values of either True or False. WARNING: The "T" or the "F" must be capitalized, or else your code will not work. They do not have quotation marks or anything special, just True or False.

All variables must be defined with an equal sign, with the variable name being *almost any name you want, as long as it's not any of these, and cannot start with a number. A equal sign represents that a variable name is assigned to a variable data.

# this is a string, our example where we have our string name, "ourfirststring" and our data, "This is some really strange text."

ourfirststring = "This is some really strange text."

By the way, if you're slightly confused on why we have a hashtag and some following text, a hashtag before any piece of text signifies that the text is just a comment, and shall be ignored when the script is running.

Examples

# Integers

somerandomintegername = -4848

# Strings

somerandomstringname = "This is a string, which I can put (almost) any piece of text inside."

# Floats

somerandomfloatname = -4848.48

# Booleans

somerandombooleanname = False
