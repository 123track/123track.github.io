Recap

Integers are all whole non-decimal numbers from negative infinity to positive infinity.

Strings are pieces of *virtually any text that have either single or double quotation marks surrounding them.

Floats are all terminating (non-repeating) numbers from negative infinity to positive infinity, including decimals.

Booleans are a True or False value, with the "T" or the "F" having to be capitalized.

The Print Function

The print function is one of the major built-in Python functions that serves a big purpose, to print text. You do not need to import any special libraries or do anything, all you need is to type print(), and an empty line of space will print out.

# the print function demo

print()

# this will print out an empty line of space

The raw print function can be used when you just need to print out a line of nothing, but if you want something meaningful to be printed, you will need to put something inside the parenthesis of the print() function. Inside the parenthesis can be any type of variable, either with its variable name or if the variable doesn't yet exist, you can just put in some variable data. Keep in mind though, that this will not create the variable for you, it will just print out some data. Any type of variable may be used in the print() function.

# printing some variables 0_0

print("You see, if it prints this, it will work, but it will not assign the string variable.")

# now i will print a boolean, integer, and float

print(True)

print(-4848)

print(-4848.48484884848848484848)

Adding print variables

You can also combine variables when using the print function, either using a comma "," or a plus sign "+". Keep in mind that if you choose to use the plus, the variables will have to be of the same type (except when using integers and floats, which are intercompatible), as it will bond together the two variables as a single variable when printing. If you choose the comma however, the variables can be of different types, but Python will automatically add a space between the variables, it treats the variables independently.

variable1 = "thisisastring"

variable2 = 384

variable3 = 3834.3

variable4 = True

# this will print variable1 and variable2 independently, but on the same line, and will automatically add a space, they can be of different types

print(variable1, variable2)

# however, the code below will throw an error because strings and integers are of different types, and we are trying to fuse them together.

print(variable1 + variable2)

# did you know that we can also combine multiple variables together in one print function with both commas and plusses?

print(variable1, variable2, variable3, variable4)

# unfortunately, this will still throw an error below because the variables are not of the same type

print(variable1 + variable2 + variable3 + variable4)

Lesson 2.2 - str(), int(), float(), bool, and the type() functions

str()

The str() function will convert any other variable type into a string. Any data type can be successfully converted into a string type, with no restrictions.

int()

The int() function will convert some variables into integers, as it is more limiting. If you want to convert a string into an integer with int(), the string must ONLY contain an integer in its contents. Floats will also technically work, but the float cannot contain any non-0 decimals. Booleans will work, returning a 1 for True and a 0 for false. Those are about the only choices that are functional.

float()

The float function is similar to the int() function, but converts data to the float type instead. The float function is able to convert variables of integers, strings (must only contain a float), integers, and booleans.

bool()

Surprisingly, the bool() function is quite permissive. It can convert all data types into a boolean. It will give a True result, as long as the variable contains data. Variables that do not contain data are: empty strings "", integers/floats with a 0.000 data, empty lists/tuples/dictionaries (we'll learn about those later), etc. If any of those are inputed, it will return a False result.

# the str() function can convert any variable into a string

print(str(True) + str(-48.2))

# the int() function is limiting

print(int(True))

# this is a demo of the float() function

print(float(482))

# bool() is very intercompatible

print(bool("38838383 this is a string actually -2949uhdJj"))

An error will be thrown if you attempt to convert a variable that does not support the conversion, for example float to integer.

type()

The type function will tell you the type of the variable if called for.

# this will print the type of our variable, variablename0

variablename0 = 383883

print(type(variablename0))
